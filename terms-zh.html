1. 服务说明

PocketClaw 是一款 iOS 远程控制应用，用于管理运行在您个人设备上的 OpenClaw 个人节点框架。本应用需要 OpenClaw Gateway 和 pocketclaw-server 插件。 PocketClaw 本身不作为公共生成式服务提供商。所有能力均由用户自行配置的本地实例提供。

2. 使用条件

使用 PocketClaw 需要： • 一台运行 OpenClaw 的个人电脑 • 安装 pocketclaw-server 插件并运行 Gateway • 通过二维码或手动设置完成设备配对 您确认仅在您被授权管理的设备上使用本应用。

3. 用户责任

• 通过 PocketClaw 远程执行的所有操作由您全权负责 • 危险操作（如重启 Gateway、重置会话、切换执行环境）需要您手动确认 • 请妥善保管您的配对 Token，不要分享给不信任的第三方 • 您应确保 OpenClaw Gateway 的安全配置符合您的安全需求
