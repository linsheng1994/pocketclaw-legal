1. 服务说明

PocketClaw 是一款第三方 iOS 客户端，用于远程管理运行在用户个人设备上的 OpenClaw 实例。OpenClaw 为独立开源项目，PocketClaw 与 OpenClaw 项目无官方隶属关系。本应用需要在您的个人电脑上安装 OpenClaw Gateway 和 pocketclaw-server 插件。 PocketClaw 本身不作为公众生成式服务提供商。所有能力均由用户自行配置的本地实例提供。连接过程中可能经由 Cloudflare Tunnel 进行加密传输，开发者不托管用户任务数据。 本应用不提供 VPN、代理或网络加速功能，不改变系统网络路由。

2. 使用条件

使用 PocketClaw 需要： • 一台运行 OpenClaw 的个人计算机 • 安装 pocketclaw-server 插件并启动 Gateway • 通过二维码或手动设置完成设备配对 您确认仅在您被授权管理的设备上使用本应用。

3. 用户责任

• 通过 PocketClaw 远程执行的所有操作由您全权负责 • 危险操作（如重启 Gateway、重置会话、切换执行环境）需要您手动确认 • 请妥善保管配对 Token，不要分享给不信任的第三方 • 您应确保 OpenClaw Gateway 的安全配置符合您的安全需求

4. 责任限制

在法律允许的最大范围内，开发者不对以下情况造成的损失承担赔偿责任： • 因 OpenClaw 框架更新导致的功能不可用 • 因网络中断或 Cloudflare Tunnel 不稳定导致的连接问题 • 因用户主动执行破坏性操作导致的数据丢失或系统异常 • 个人设备上运行的第三方应用的输出内容 • 因不可抗力（包括但不限于自然灾害、政府行为、互联网基础设施故障）导致的服务中断 本协议不排除或限制适用法律不允许排除或限制的责任。

您可以随时停止使用 PocketClaw 并卸载 App。卸载后，所有应用数据（沙盒数据和 App Group 共享存储）将被系统自动删除。
