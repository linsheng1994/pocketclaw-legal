🔒 Privacy Policy

Last updated: March 12, 2026

Overview

PocketClaw is an iOS third-party client for remotely managing OpenClaw instances running on your personal computer. We take your privacy very seriously. This Privacy Policy explains how PocketClaw collects, uses, and protects your information. Special note: This app does not provide generative AI services to the public, does not host model inference services. All intelligent capabilities are provided by user-owned device instances. Connections may be relayed via Cloudflare Tunnel for encrypted transmission; the developer does not host user task data. This app does not provide VPN, proxy, or network acceleration services, and does not alter system network routing.

1. Data Collection & Storage

PocketClaw only stores the following data on your local device: • Node connection info (host address, port, node name) — stored in UserDefaults • Authentication token — stored in App Group shared storage (UserDefaults), accessible by the app and its Widget extension • Task execution history (last 50 records) — stored in app sandbox files (Application Support) • User preferences — stored in UserDefaults The developer does not operate or host any user data servers. Network requests may be relayed through Cloudflare infrastructure; the developer does not receive or store task content.

2. Network Communication

PocketClaw communicates with the OpenClaw Gateway on your personal computer using the following methods: • Cloudflare Tunnel: Establishes an encrypted tunnel via Cloudflare infrastructure for remote access. Data is relayed through Cloudflare servers but encrypted with HTTPS. When using Cloudflare Tunnel, encrypted data may transit through Cloudflare nodes located outside your country. Cloudflare processes transit data per its privacy policy; the developer does not access or retain transmission content. • LAN Direct: When Tunnel is unavailable, the App can connect to Gateway directly via local network (HTTP). LAN direct connection is intended only for trusted local network environments; all public network access uses HTTPS Tunnel. All API requests carry Bearer Token authentication. All requests target your own OpenClaw Gateway exclusively. The developer does not operate any intermediate servers and does not receive user request content. In LAN direct-connect scenarios, iOS may prompt for "Local Network" permission; this permission is only used to connect to your authorized personal gateway device.

3. Camera Usage

PocketClaw only requests camera permission when scanning the pairing QR code. Camera data is not recorded, stored, or transmitted. You can revoke camera permission at any time in iOS System Settings.

4. Notification Permission

PocketClaw may request notification permission when you enable "Task Result Notifications". Notifications are used to alert you when tasks complete, fail, or need attention. Notification content is used solely for local reminders — never for advertising, profiling, or cross-app tracking. You can disable notification permission at any time in iOS System Settings.

5. Third-Party Services

• Cloudflare: Used for Tunnel relay (established by cloudflared on the host). Cloudflare's privacy policy: https://www.cloudflare.com/privacypolicy/ • PocketClaw does not integrate any analytics SDK, advertising SDK, or user tracking services.

6. Data Security

• Authentication tokens are stored in App Group shared storage with iOS data protection • Remote communication is encrypted via HTTPS (Cloudflare Tunnel) • Sensitive operations (e.g., restart service, reset session) require manual user confirmation • The App does not collect device identifiers (IDFA/IDFV)

7. Data Deletion & Your Rights

You can delete saved node information at any time in Settings → Node Management. After uninstalling the App, all sandbox data and App Group data (including stored tokens) will be automatically deleted by the system. You have the right to access, correct, or delete your personal data. To exercise these rights, please contact us using the information provided below.

8. Children's Privacy

PocketClaw is not intended for children under 13 and does not knowingly collect personal information from children.

9. Policy Updates

We may update this Privacy Policy from time to time. Updated policies will be displayed within the App. Continued use of the App indicates your acceptance of the updated policy.
