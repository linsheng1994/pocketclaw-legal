🔒 隐私政策

最后更新：2026 年 3 月 12 日

概述

PocketClaw 是一款 iOS 遥控器应用，用于远程管理运行在您个人电脑上的 OpenClaw 框架。我们非常重视您的隐私。本隐私政策说明了 PocketClaw 如何收集、使用和保护您的信息。 特别说明：本应用不对公众提供生成式 AI 服务，不托管模型推理服务。所有智能能力均由用户自有设备实例提供。PocketClaw 是 OpenClaw 框架的官方配套应用，由同一开发团队开发。

1. 数据收集与存储

PocketClaw 仅在您的本地设备上存储以下数据： • 节点连接信息（主机地址、端口、节点名称）— 存储在 UserDefaults • 认证令牌（Token）— 存储在 App Group 共享存储（UserDefaults App Group），供主应用和 Widget 扩展共同访问 • 任务执行历史（最近 50 条记录）— 存储在应用沙盒文件（Application Support） • 用户偏好设置 — 存储在 UserDefaults 所有数据仅存储在您的 iPhone 本地，不会上传至任何云端服务器。

2. 网络通信

PocketClaw 通过以下方式与您个人电脑上的 OpenClaw Gateway 通信： • Cloudflare Tunnel：通过 Cloudflare 基础设施建立加密隧道进行远程访问。数据经 Cloudflare 服务器中转，但全程以 HTTPS 加密。 • 局域网直连：当 Tunnel 不可用时，App 可通过本地网络（HTTP）直接连接 Gateway。 所有 API 请求均携带 Bearer Token 认证。所有请求仅指向您自己的 OpenClaw Gateway。在 Tunnel 模式下，网络流量经 Cloudflare 基础设施中转。开发者不运营任何中间服务器，不接收用户请求内容。 在局域网直连场景中，iOS 可能会提示"本地网络"权限；此权限仅用于连接您已授权的个人网关设备。

3. 摄像头使用

PocketClaw 仅在扫描配对二维码时请求摄像头权限。摄像头数据不会被录制、存储或传输。您可以随时在 iOS 系统设置中撤销摄像头权限。

4. 通知权限

当您开启"任务结果通知"时，PocketClaw 可能会请求通知权限。通知用于在任务完成、失败或需要注意时提醒您。通知内容仅用于本地提醒，绝不用于广告、用户画像或跨应用追踪。您可以随时在 iOS 系统设置中关闭通知权限。

5. 第三方服务

• Cloudflare：用于 Tunnel 中继（由宿主机上的 cloudflared 建立）。Cloudflare 隐私政策：https://www.cloudflare.com/privacypolicy/ • PocketClaw 未集成任何分析 SDK、广告 SDK 或用户追踪服务。

6. 数据安全

• 认证令牌存储在 App Group 共享存储中，受 iOS 数据保护机制保障 • 远程通信通过 HTTPS（Cloudflare Tunnel）加密 • 敏感操作（如重启服务、重置会话）需要用户手动确认 • App 不收集设备标识符（IDFA/IDFV）

7. 数据删除

您可以随时在设置 → 节点管理中删除已保存的节点信息。卸载 App 后，所有沙盒数据和 App Group 数据（包括存储的 Token）将被系统自动删除。

8. 儿童隐私

PocketClaw 不面向 13 岁以下的儿童，也不会故意收集儿童的个人信息。

9. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在 App 内显示。继续使用 App 即表示您接受更新后的政策。
