1. Service Description

PocketClaw is a third-party iOS client for remotely managing OpenClaw instances running on your personal device. OpenClaw is an independent open-source project; PocketClaw has no official affiliation with the OpenClaw project. This app requires OpenClaw Gateway and the pocketclaw-server plugin to be installed on your personal computer. PocketClaw does not serve as a public generative service provider. All capabilities are provided by user-configured local instances. Connections may be relayed via Cloudflare Tunnel for encrypted transmission; the developer does not host user task data. This app does not provide VPN, proxy, or network acceleration services, and does not alter system network routing.

2. Usage Conditions

To use PocketClaw, you need: • A personal computer running OpenClaw • The pocketclaw-server plugin installed and Gateway running • Device pairing completed via QR code or manual setup You confirm that you will only use this app on devices you are authorized to manage.
